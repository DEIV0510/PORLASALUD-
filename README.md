# IPS PORLASALUD — Landing Page

Landing page corporativa premium para **IPS PORLASALUD**, empresa de salud colombiana.
Slogan: _"Al cuidado de la familia"_.

> Salud domiciliaria · Traslados asistenciales básicos y medicalizados · Ambulancia aérea ·
> Pediatría · Fisioterapia · Nutrición · Atención al adulto mayor.

## ✨ Características

- **Pantalla de carga** con logo, pulso cardíaco animado, barra de progreso y partículas.
- **Hero** a pantalla completa con imagen institucional y llamados a la acción.
- **8 servicios** en tarjetas premium con foto, ícono y contacto directo por WhatsApp.
- **Estadísticas animadas** (contadores) y sección "¿Por qué elegirnos?".
- **Mapa de cobertura de Colombia** interactivo con rutas médicas animadas.
- **Galería** con lightbox, testimonios y equipo profesional.
- **Formulario de contacto** que genera el pedido y lo envía por WhatsApp.
- **WhatsApp flotante**, botón "volver arriba" y navegación responsive.
- 100% **responsive** (escritorio, tablet, móvil) y optimizado.

## 🛠️ Tecnologías

- HTML5, CSS3 y JavaScript (vanilla, sin frameworks ni Bootstrap)
- [GSAP](https://gsap.com/) + ScrollTrigger
- [ScrollReveal](https://scrollrevealjs.org/)
- [AOS](https://michalsnik.github.io/aos/)
- Fuentes: Sora + Inter (Google Fonts)
- Imágenes en formato WebP

## 🚀 Cómo ejecutar

Es un sitio estático. Puedes abrir `index.html` directamente o levantar el servidor incluido:

```bash
node serve.js
# → http://localhost:5228
```
